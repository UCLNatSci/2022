Project (LSA Alternative Assessment)#

Introduction#

In Section 5 we studied cellular automaton models, which consist of a grid of cells with assigned states. The state of each cell is updated at discrete time steps according to the state of neighbouring cells. Such models have been applied to study a variety of physical systems including crystal growth, traffic flow, flocking behaviour and chemical reactions.

A cellular automaton can be used to model the spread of an epidemic. A population is modelled as a grid of cells, where each cell is assumed to represent an individual person. Each cell is labelled according to its current state of infection, and the state of each cell updates at every timestep according to a set of probabilistic rules.

Question 1 [20 marks]#

In the `SIR’ model, each grid cell is in one of three states.

State

Description

S

Susceptible

I

Infected

R

Recovered

Susceptible cells have yet to contract the virus but may change to infected if a neighbouring cell is infected. Infected cells currently have the virus but may change to recovered. Recovered cells are immune and so will not change state. The model is governed by two parameters \(p_\mathrm{i}\) (probability of infection) and \(p_\mathrm{r}\) (probability of recovery). At each timestep, the following rules are applied to each cell in the grid.

  1. If the cell is S and any of its 8 immediate neighbours are I then the cell becomes I with probability \(p_\mathrm{i}\), otherwise it remains S.

  2. If the cell is I then it becomes R with probability \(p_\mathrm{r}\), otherwise it remains I.

  3. If the cell is R then it remains R.

You should Write a cellular automoton model which implements the rules described above for a \(10\) by \(10\) grid with \(p_i = 0.1\) and \(p_r = 0.2\). Assume that initially every grid cell is S except for one cell (located near the centre of the grid) which is I. Execute your simulation for \(20\) timesteps and display the grid at each timestep. Then, display a line graph showing the total number of cells in states S, I and R at each timestep.

At each timestep, your model should calculate a new grid based on the values of the grid in the previous timestep. It is suggested that you base your answer on the material in the Section 5.3.2.

Question 2 [30 marks]#

Use a cellular automaton model to investigate the spread of an epidemic, and write up your investigation as a Jupyter Notebook. You should use your answer to Question 1 as a starting point, but you could also investigate more sophisticated models (such as SEIR or SEIQR). Some questions you might like to investigate are:

  • How does the spread of the epidemic depend on the initial distribution of infections?

  • How does the spread of the epidemic depend on parameters \(p_i\), \(p_r\) or the size of the grid?

  • What is the most effective way to stop the spread of the epidemic (eg introducing quarantine or reducing the infection probability)?

You do not need to investigate all the questions above - it is likely better to investigate one or two questions thoroughly. Alternatively, you might like to try to reproduce one of the results from this paper.

You should submit a single Jupyter Notebook which is both a working solution to the scientific problem and a report which documents the solution. It is up to you how you organise the notebook, however your notebook should be a coherent and self-contained document which:

Introduces and motivates the problem. There should be sufficient background information that the reader can understand your report without needing to refer elsewhere.

Clearly sets out the solution to the problem using text, illustrations and working code. Aim to produce writing of the same standard as a scientific report, using appropriate headings to provide structure and figures and neatly formatted equations where appropriate. Enough detail should be included such that the reader could reproduce your solution and adapt it for their own purposes.

Runs without errors on the Cocalc platform. A penalty will be applied to submissions which produce errors or don’t complete in a timely fashion.

Notes#

  • You may divide your submission into multiple code files, however you should include only one .ipynb file which presents the solution to the problem.

  • You may only use only libraries pre-installed in Cocalc. Do not install any other third-party libraries.

Submission#

You should submit one notebook file for Question 1 and another notebook file for Question 2. Place your work in the folder NSCI0007_GROUP_PROJECT in your Cocalc project. This folder will be automatically collected at the end of the assessment period.
