2.1. Modelling Change¶

Last week we developed code that simulated exponential growth or decay. This week we will take this a step further by storing the results of the simulation in an array which is a Python data type for storing a sequence of numerical values. This will then allow us to plot graphs of the results of the simulation.

2.1.1. Importing the numpy library¶

The standard Python language is very extensive, containing a wide range of built-in data types and functions, some of which we looked at last week. However, in addition to the standard Python language, we can access an ever-growing collection of code libraries. A library is a bundle of code that provides commonly used functions and objects. One of the most important for scientists is the numpy library which contains the array data type and a number of useful mathematical functions. We will be using this library extenstively for the remainder of the course.

import numpy as np

x = np.sqrt(2)

print("The square root of 2 is", x)

The square root of 2 is 1.4142135623730951

The line import numpy as np instructs Python to load the numpy library and make it available using the prefix np..

2.1.2. Arrays¶

An array is a Numpy data type that stores a sequence of values. For example, suppose we have the following data representing five years’ population data:

populations = np.array([12, 25, 54, 102, 206])
print(populations)

[ 12  25  54 102 206]

Each element in an array has an index which identifies its position. We can access an array element by following the variable name with square brackets and the index:

initial_pop = populations[0]
year_3_pop = populations[2]

print("Initial population:", initial_pop)
print("Population in year 3:", year_3_pop)

Initial population: 12
Population in year 3: 54

Note

Arrays are indexed starting from 0, so the first element of array x is x[0], the second is x[1] and so on. If an array has n elements, then the last element is at index n - 1.

Likewise, we can update individual array elements. Suppose we would like to change the value of the 3rd element:

populations[2] = 100
print(populations)

[ 12  25 100 102 206]

2.1.3. Creating Numpy Arrays¶

Above, we created an array using the np.array function. Other ways to create an array include the function np.zeros and np.linspace.

z = np.zeros(6) # Create an array of 6 zeros
print(z)

[0. 0. 0. 0. 0. 0.]

Use np.linspace(a, b, num) To generate num evenly spaced numbers between a and b:

z = np.linspace(0, 1, 4)
print(z)

[0.         0.33333333 0.66666667 1.        ]

2.1.4. for Loops¶

A for loop is used to iterate over a sequence of values. The simplest form of the for statement is to iterate over a range of n integer values. For example, the following code will execute the indented statements once for each integer from 0 to 4:

for i in range(5):
    print("i =", i)

i = 0
i = 1
i = 2
i = 3
i = 4

The variable i is called the iterator variable and is updated during each iteration.

The range function specifies the sequence of integer values that the loop variable will take.

range(n) generates a sequence from 0 to n - 1.

range(a, n) generates a sequence from a to n - 1.

range(a, n, step) generates a sequence from to n - 1 in steps of step.

for i in range(1, 10, 2):
    print("i =", i)

i = 1
i = 3
i = 5
i = 7
i = 9

Exercise 2.4

Write a for loop which prints the following:

i = 2
i = 5
i = 8
i = 11

Solution to Exercise 2.4

The first value is 2, so a = 2; the final value is 11 so n - 1 = 11, therefore n = 12; the values increase in steps of 3 so step = 3, so we need:

for i in range(2, 12, 3):
    print("i =", i)

We plot graphs using a Python package called matplotlib.

import matplotlib.pyplot as plt

dis = np.array([2.5, 2.6, 3.4, 4.1, 4.5, 5.1, 5.2])

plt.figure(figsize=(4,4)) # create figure of size 4 by 4 inches
plt.plot(dis) # plot a line graph

[<matplotlib.lines.Line2D at 0x26816db9808>]

First we have to import the plotting package which enables us to access plotting functions using the prefix plt.

Next, we create an array dis containing the values to plot.

plt.figure(figsize=(4,4)) creates a figure of the given size.

Finally, plt.plot(dis) plots the values on a line graph.

Notice that the data points are plotted at x-coordinates 0 ... 6. To use the actual Time values, we need to create a second array time and pass both arrays to the function plt.plot.

dis = np.array([2.5, 2.6, 3.4, 4.1, 4.5, 5.1, 5.2])
time = np.linspace(0.1, 0.7, 7)

plt.figure(figsize=(4,4))
plt.plot(time, dis)

[<matplotlib.lines.Line2D at 0x26818fb0e08>]

Finally, let’s add a title and axis labels using functions plt.xlabel, plt.ylabel and plt.title.

plt.figure(figsize=(4,4)) 
plt.plot(time, dis)
plt.xlabel("Time (s)") # add an x-axis label
plt.ylabel("Distance (m)") # add a y-axis label
plt.title("Particle Position") # add a figure title

Text(0.5, 1.0, 'Particle Position')
